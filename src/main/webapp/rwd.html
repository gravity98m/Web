rwd(Responsive Web Design)

반응형웹. 조건에 따라 알맞은 스타일을 적용하도록 함.
Media Queries로 구현(모바일퍼스트가 원칙~)

Viewport 설정(맞추기)

<meta name="viewport" content="width=device-width, initial-scale=1.0">
모바일에서 화면 축소 방지

Media Queries구문

@media not|only 장치종류 and (조건) { 선택자 {속성:속성값;} }

Media Queries 장치 종류 조건

  • all - 장치 종류 무관
  • print - 프린터
  • screen - 컴퓨터(모바일기기 포함)
  • speech - 음성낭독기

Media Queries 기타 조건

  • width and height of the viewport - 화면 너비나 높이
  • width and height of the device - 장치 너비나 높이
  • orientation (landscape/portrait) - 화면방향(가로/세로)
  • resolution - 해상도
