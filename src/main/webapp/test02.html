html(웹문서)을 구성하는 필수 태그들

html ... /html - 웹 문서의 시작과 끝을 선언
head ... /head - 머리말 영역의 시작과 끝을 선언
title ... /title - 제목의 시작과 끝을 선언
body ... /body - 본문의 시작과 끝을 선언

웹문서의 구조

    <html>		- 웹 문서의 시작
     <head> 		- 머리말 영역의 시작
      <title> ... </title>	- 제목의 시작과 끝
     </head>		- 머리말 영역의 끝
     <body>		- 본문의 시작

     </body>		- 본문의 끝
    </html>		- 웹 문서의 끝
