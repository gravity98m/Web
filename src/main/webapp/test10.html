iframe으로 페이지 내에 다른 페이지 보여주기

<iframe src="보여줄..." width="n" heigt="n"></iframe>

iframe안에 보여줄 수 있는 종류들

  • URL
  • web page - *.html or *.htm
  • image or movie - *.jpg/*.gif/*.png/*.mp4/*.webm/*.ogg
  • text file - *.txt
