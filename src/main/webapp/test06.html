form으로 사용자와 연결하기

form요소 - input type의 값들

  1. input type="text" 주관식. 한 줄 글상자
  2. (input에 대한 설명문을 뜻하는 label.
    label태그와 input태그를 for와 id값으로 서로 연결시켜 줌)
    size="n"으로 입력창의 길이를 지정.
    maxlength="n"으로 입력값의 최대치를 제한.
  3. input type="password" 주관식. 암호 글상자
  4. textarea ... .textarea 주관식. 큰 글상자
    cols="n"와 rows="n"로 상자의 가로크기와 세로크기를 지정.
  5. input type="radio" 객관식(단일선택형). 라디오버튼

    (같은 질문 하의 요소끼리는 name의 값을 서로 일치시킴) 초기값으로 지정된 요소는 checked(checked="checked")를 추가.

    가장 선호 색상은?

  6. input type="checkbox" 객관식(다중선택형).

    (같은 질문 하의 요소끼리는 name의 값을 서로 일치시킴) 초기값으로 지정된 요소는 checked(checked="checked")를 추가.

    선호 색상은?

  7. select ... /select 공간절약향 객관식. 펼침목록(드롭다운리스트)
    select안에 option으로 항목들을 추가함. selected속성으로 초기값을 지정하고

    가장 선호 색상은?

    가장 선호 색상은? (selected="selected"/selected)

    상담 가능한 시간은? (다중선택 multiple="multiple"/multiple)

    상담 가능한 시간은? (상자형 size="n" 추가)

    optgroup태그로 서로 관련있는 option들을 묶어서 그룹화하여 나타낼 수 있고
    해당 그룹의 주제, 제목 등은 label속성으로 추가.

    오전과 오후 중 각 상담 가능한 시간은?

  8. input type="file" 파일전송용 상자
  9. input type="hidden" 보이지 않는 정보 전송용
  10. input type="time" 시간(오전/오후 시,분)을 지정
  11. input type="date" 날짜(년,월,일)을 지정
    min속성으로 최소값을, max속성으로 최대값
  12. input type="datetime-local" 날짜와 시간(년,월,일,오전/오후,시,분)을 지정
  13. input type="week" 날짜(년,주)을 지정
  14. input type="month" 날짜(년,월)을 지정
  15. input type="number"
    min속성으로 최소값을, max속성으로 최대값, step속성으로 증감 간격 지정

    권(min="0")
    권(max="10")
    (step="12")
  16. input type="range" 범위 내의 값을 지정
    min속성으로 최소값을, max속성으로 최대값, step속성으로 증감 간격 지정
  17. input type="tel" 연락처(전화번호)를 지정.
    가상키패드에서 숫자 형식으로 바뀔 수 있고 전화 걸기 가능
  18. input type="email" 연락처(이메일)를 지정
    가상키패드에서 @나 영문 형식으로 바뀔 수 있고 웹브라우저가 알맞은 email형식인지 체크해줌(@유무 등)
  19. input type="url" 연락처(URL, 웹주소)를 지정
    가상키패드에서 .com나 영문 형식으로 바뀔 수 있고 웹브라우저가 알맞은 웹주소형식인지 체크해줌(http://, https://유무 등)
  20. input type="color"
    입니다.
  21. input type="search" 검색창
    삭제 버튼(아이콘)이 추가됨

html4에서 추가된 속성들

  1. autofocus(autofocus="autofocus")
  2. autocomplete="on/off" 자동완성 기능을 켜거나 끔
  3. required(required="required") 필수입력 요소로 지정
  4. placeholer="***" 입력예시나 설명문 등 표시
  5. readonly(readonly="readonly") 읽기만 가능한 요소로 지정
    (입력값이 전송 됨)
    입니다.
  6. disabled(disabled="disabled") 사용불가능한 요소로 지정
    (입력값이 전송 안됨)
    입니다.
